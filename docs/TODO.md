## Критичное (ядро геймплея)

- [x] **Матчинг черт по имени ломался под RU (root-баг).** Имена черт/боевых
  стилей сохранялись в `character.feats[].name` переведёнными ("Защита"), а
  логика статов матчила по английским ключам. Исправлено: добавлен стабильный
  ключ `feat.nameEn` (английский), вся логика статов матчит по `nameEn ?? name`,
  отображение осталось локализованным, + миграция существующих персонажей
  (бэкфилл `nameEn` + пересчёт КД/инициативы) в [CharacterSheet.tsx](../src/components/CharacterSheet.tsx).
  Остаточно: то же самое нужно для воззваний (`optionalFeatures`, напр.
  `Agonizing Blast` в [spellCasting.ts](../src/utils/spellCasting.ts)) — пока матчит по локализованному имени.

- [x] **Отдых (короткий/длинный).** Кнопки в панели «Здоровье»: длинный — полное
  HP, сброс temp HP, возврат половины костей хитов, сброс ячеек/ресурсов/
  спасбросков от смерти; короткий — ресурсы короткого отдыха + ячейки Пакта
  колдуна. Восстановление по `restoreOn` (новый `isShortRestResource`).
- [x] **Лечение костями хитов (ручное на коротком отдыхе).** Кнопка «Лечить» в
  панели «Здоровье»: бросок кости хитов + Тел, восстановление HP (мин. 1, не выше
  максимума), списание кости. См. `rollHitDie` в [CharacterSheet.tsx](../src/components/CharacterSheet.tsx).
- [x] **Состояния дают механику.** Новый [conditionEffects.ts](../src/utils/conditionEffects.ts):
  `Grappled`/`Restrained`/`Paralyzed`/`Petrified`/`Stunned`/`Unconscious` обнуляют
  скорость (учитывается в эффективной скорости в сайдбаре), у каждого состояния на
  чипе показан список механических эффектов (помехи/преимущества/провалы спасбросков
  и т.д.). Помехи/преимущества на конкретные d20 не подставляются автоматически —
  лист не катит атаки за игрока.
- [x] **Истощение (Exhaustion).** Поле `exhaustion` (0–6), степпер в панели
  «Здоровье». Правила 2024: −2×уровень ко всем проверкам d20 (навыки, спасброски,
  атаки, инициатива, пасс. Внимательность) и −5 фт×уровень к скорости.
- [ ] **Мультикласс.** Поддержан только один класс/уровень.

## Среднее

- [x] **Боевые стили в боевой математике.** Archery (+2 к меткости дальнего боя),
  Dueling (+2 к урону одноручным без второго оружия), Two-Weapon Fighting (мод
  урона второй руки только со стилем), Thrown Weapon Fighting (+2 к урону при
  метании). Great Weapon Fighting (переброс 1–2 не выражается числом) показан
  маркером «БО/GWF» на подходящих атаках. См. [weaponAttacks.ts](../src/utils/weaponAttacks.ts).
- [x] **Extra Attack.** Бейдж «N× атаки» в секции атак. `getAttacksPerAction`:
  Воин (2/3/4 на 5/11/20), варвар/монах/паладин/следопыт (2 на 5), подклассы
  (Мастер меча и т.д.), Воззвания колдуна Thirsting/Devouring Blade. См.
  [weaponAttacks.ts](../src/utils/weaponAttacks.ts).
- [x] **Лимит настройки (3 предмета).** Equip 4-го настраиваемого предмета
  надевает его без настройки (бонусы не применяются) + счётчик «Настройка N/3».
  См. [InventoryGrid.tsx](../src/components/InventoryGrid.tsx).
- [x] **Power-attack toggles.** Переключатель «−5/+10» в секции атак (виден при
  наличии Sharpshooter / Great Weapon Master). SS применяется к дальнему оружию,
  GWM — к тяжёлому рукопашному; помеченные атаки показывают изменённые меткость/урон.
- [x] **Героическое вдохновение.** Поле `heroicInspiration`, кнопка-звезда в
  панели «Здоровье».
- [x] **Чувства (тёмное зрение/blindsight).** Поле `senses` (тёмное/слепое зрение,
  чувство вибрации, истинное зрение). Редактор в листе («Чувства») + блок в
  сайдбаре. Авто-заполнение из вида пока вручную.
- [x] **Нагрузка/грузоподъёмность.** Порог Сил×15 (с учётом предметов на Силу)
  показан рядом с весом; при превышении — индикатор «Перегрузка». См. [InventoryGrid.tsx](../src/components/InventoryGrid.tsx).
- [x] **Экономика.** Покупка (списание стоимости с монет при добавлении, проверка
  достаточности) и продажа (за полцены). Конвертер монет в [currency.ts](../src/utils/currency.ts).
- [x] **Концентрация.** Поле `concentratingOn`, секция «Концентрация»: текущее
  заклинание + спасбросок концентрации (Сл = max(10, урон/2), бросок 1d20 + спас.
  Тел). См. [CharacterSheet.tsx](../src/components/CharacterSheet.tsx).

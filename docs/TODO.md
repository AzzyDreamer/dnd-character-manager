## Критичное (ядро геймплея)

- [ ] **Матчинг черт по имени ломается под RU (root-баг).** Имена черт/боевых
  стилей сохраняются в `character.feats[].name` переведёнными ("Защита"), а вся
  логика статов матчит по английским ключам (`FEAT_STAT_EFFECTS['Defense']`,
  `f.name === 'Alert'`, `'Medium Armor Master'`). Под RU (язык по умолчанию для
  RU-навигатора) это не срабатывает — т.е. **уже сделанные** эффекты (Defense +1
  КД, Squat Nimbleness, Dragon Hide, MAM, Alert-инициатива) под RU не работают.
  Правильный фикс: хранить английское имя (`_origName`) в `character.feats`,
  переводить только при отображении, + миграция существующих персонажей. Это
  предпосылка для «боевых стилей в бою».

- [ ] **Отдых (короткий/длинный).** Кнопок отдыха нет; метаданные
  `restoreOn: 'short'|'long'` у ресурсов ([classResources.ts](../src/utils/classResources.ts))
  не используются. Нужно: восстановление HP, ячеек заклинаний, ресурсов класса,
  костей хитов, сброс спасбросков от смерти, снятие 1 уровня истощения (long).
- [ ] **Лечение костями хитов.** Сейчас трата кости только уменьшает счётчик,
  HP не восстанавливаются (короткий отдых).
- [ ] **Состояния дают механику.** `Prone`, `Poisoned`, `Grappled` (скорость 0),
  `Restrained`, `Frightened` и т.д. сейчас чисто косметические метки
  (`character.conditions: string[]`). Нет помех/преимуществ/обнуления скорости.
- [ ] **Истощение (Exhaustion).** Не отслеживается (нет поля). Правила 2024:
  −2×уровень ко всем проверкам d20 и −5 фт×уровень к скорости.
- [ ] **Мультикласс.** Поддержан только один класс/уровень.

## Среднее

- [ ] **Боевые стили в боевой математике.** Defense (+1 КД) подключён, но Archery
  (+2 к меткости), Dueling (+2 к урону), Two-Weapon Fighting (мод ко второй
  руке), Great Weapon Fighting (переброс 1–2) инертны. См. [weaponAttacks.ts](../src/utils/weaponAttacks.ts).
- [ ] **Extra Attack.** Число атак за действие «Атака» не отображается.
- [ ] **Лимит настройки (3 предмета).** Не enforced — можно настроиться на любое
  число магических вещей. См. [InventoryGrid.tsx](../src/components/InventoryGrid.tsx).
- [ ] **Power-attack toggles.** Sharpshooter / Great Weapon Master (−5/+10) — нет
  переключателя в атаках.
- [ ] **Героическое вдохновение.** Не трекается.
- [ ] **Чувства (тёмное зрение/blindsight).** Нет поля у `Character`, данные
  вида/черт по чувствам игнорируются; на листе не показаны.
- [ ] **Нагрузка/грузоподъёмность.** Общий вес считается, но нет порога (Сил×15)
  и штрафов.
- [ ] **Экономика.** Предметы добавляются бесплатно, золото не списывается; нет
  покупки/продажи.
- [ ] **Концентрация.** Нет состояния «концентрируюсь на …» и спасброска
  концентрации при уроне (у заклинаний только метаданные).
